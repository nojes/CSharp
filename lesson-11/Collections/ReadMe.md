# Collections

## **Generic**

   Template classes
   ```csharp
   List<T> ...
   ```
   
## **Not generic**
    
Packaged :arrow_down:

Cast to `object` type
```csharp 
object int a = 15;
object o = (object)a;
```

#### **Sequence**
* `ArrayList` - indexed.
* `Stack` - reverse extract (LIFO).
* `Queue` - direct extract (FIFO).

#### **Associative**
* `SortedList`
* `HashTable`
